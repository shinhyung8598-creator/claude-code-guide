# 마크다운 문법 총정리

마크다운은 읽기 쉽고 쓰기 쉬운 텍스트 형식입니다. 자주 쓰는 문법을 한 곳에 정리합니다.

## 제목

```
# H1
## H2
### H3
```

## 강조

- **굵게** → `**굵게**`
- *기울임* → `*기울임*`
- ~~취소선~~ → `~~취소선~~`
- `인라인 코드` → `` `인라인 코드` ``

## 목록

순서 없는 목록:

- 사과
- 바나나
  - 노란 바나나
  - 초록 바나나
- 체리

순서 있는 목록:

1. 첫 번째
2. 두 번째
3. 세 번째

## 인용구

> 단순함은 궁극의 정교함이다.
> — 레오나르도 다 빈치

## 코드 블록

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```
