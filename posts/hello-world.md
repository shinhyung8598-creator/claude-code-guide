# Hello World — 블로그를 시작하며

오늘부터 글을 씁니다. 이 블로그는 **순수 HTML, CSS, JavaScript**만으로 만들었습니다. 빌드 도구도, 프레임워크도 없습니다.

## 왜 직접 만들었나

기존 블로그 플랫폼은 기능이 많지만 그만큼 무겁습니다. 마크다운 파일 하나를 웹에서 읽을 수 있으면 충분한데, 수백 MB의 `node_modules`가 필요하지는 않죠.

## 작동 방식

1. `posts/index.json`에 포스트 메타데이터를 작성한다.
2. `posts/` 폴더에 `.md` 파일을 추가한다.
3. 로컬 HTTP 서버를 켜고 브라우저로 접속한다.

```bash
python -m http.server 8000
```

그게 전부입니다.
