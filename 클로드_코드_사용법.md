# 클로드 코드(Claude Code) 사용법 가이드
> 코딩을 몰라도 괜찮아요! 이 가이드는 처음 시작하는 분을 위해 작성되었습니다.

---

## 1. 클로드 코드란?

클로드 코드는 **AI와 대화하면서 코딩 작업을 맡기는 도구**입니다.
직접 코드를 짤 필요 없이, 하고 싶은 것을 말로 설명하면 AI가 대신 만들어 줍니다.

---

## 2. 설치 방법

### 방법 A: 데스크톱 앱 (가장 쉬움)
1. [claude.ai](https://claude.ai) 에 접속해서 계정을 만드세요.
2. Claude Code 데스크톱 앱을 다운로드해서 설치하세요.
3. 로그인하면 바로 사용 가능합니다.

### 방법 B: VS Code 확장 프로그램
1. VS Code를 설치합니다.
2. 확장 프로그램 탭에서 "Claude Code"를 검색해서 설치합니다.

### 방법 C: 터미널(명령줄)로 설치
```
npm install -g @anthropic-ai/claude-code
```
> 이 방법은 Node.js가 먼저 설치되어 있어야 합니다.

---

## 3. 기본 사용법

### 대화하듯이 요청하세요

클로드 코드는 **채팅창에 원하는 것을 입력**하면 됩니다.
어렵게 생각하지 말고, 사람에게 부탁하듯이 말하면 됩니다.

**예시 요청:**
- "파이썬으로 간단한 계산기를 만들어줘"
- "이 코드에서 오류가 나는데 고쳐줘"
- "index.html 파일을 읽어서 어떤 내용인지 설명해줘"
- "폴더 안에 있는 파일 목록을 보여줘"

---

## 4. 자주 쓰는 슬래시 명령어

채팅창에 `/`를 입력하면 특별한 명령어를 사용할 수 있습니다.

| 명령어 | 하는 일 |
|--------|---------|
| `/help` | 도움말 보기 |
| `/clear` | 대화 내용 초기화 |
| `/config` | 설정 변경 |
| `/code-review` | 코드 검토 요청 |
| `/run` | 앱 실행 |

**사용 예시:** 채팅창에 `/help` 라고 입력하고 엔터를 누르면 됩니다.

---

## 5. 작업 폴더 지정하기

클로드 코드는 **내가 작업하는 폴더를 기준**으로 파일을 읽고 수정합니다.

- 앱을 실행할 때 작업하고 싶은 폴더를 열어두면 됩니다.
- "이 폴더에서 작업해줘" 라고 말해도 됩니다.

---

## 6. 파일 다루기

### 파일 읽기
> "project 폴더 안에 있는 main.py 파일을 읽어줘"

### 파일 만들기
> "Hello World를 출력하는 파이썬 파일을 만들어줘"

### 파일 수정하기
> "index.html의 제목을 '내 첫 번째 웹사이트'로 바꿔줘"

---

## 7. 유용한 활용 예시

### 웹사이트 만들기
> "간단한 포트폴리오 웹페이지를 HTML로 만들어줘. 내 이름은 홍길동이고, 소개 문구도 넣어줘."

### 자동화 스크립트 만들기
> "특정 폴더의 사진 파일 이름을 날짜 순서대로 바꿔주는 프로그램을 만들어줘"

### 오류 해결하기
> "이 오류 메시지가 뭔 뜻이야? 어떻게 고쳐?" (오류 내용을 붙여넣기)

### 코드 설명 듣기
> "이 코드가 어떻게 동작하는지 초등학생도 이해할 수 있게 설명해줘"

---

## 8. 대화할 때 팁

| 잘 되는 방식 | 피해야 할 방식 |
|-------------|--------------|
| 구체적으로 설명하기 | 너무 막연하게 말하기 |
| 원하는 결과물 설명하기 | 과정만 말하기 |
| 모르는 것은 그냥 물어보기 | 혼자 추측하기 |

**좋은 요청 예시:**
> "파이썬으로 숫자를 입력하면 짝수인지 홀수인지 알려주는 프로그램을 만들어줘. 파일 이름은 check.py로 해줘."

**아쉬운 요청 예시:**
> "프로그램 만들어줘."

---

## 9. 알아두면 좋은 것들

- **실수해도 괜찮아요.** 잘못됐다 싶으면 "아니야, 다시 해줘" 라고 하면 됩니다.
- **이전 대화를 기억해요.** 같은 대화 안에서는 앞서 한 얘기를 기억합니다.
- **파일을 직접 수정해요.** 클로드 코드는 실제 파일을 바꾸니까, 중요한 파일은 백업해두세요.
- **모르는 단어가 나오면 물어보세요.** "방금 한 말에서 ○○○이 뭐야?" 라고 바로 물어보면 됩니다.
